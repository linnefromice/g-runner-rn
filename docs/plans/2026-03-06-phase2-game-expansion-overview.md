# Phase 2: ゲーム拡張 — 全体計画

## 目的

v0.1.0 の基盤の上に、ゲームとしての面白さ・深み・リプレイ性を大幅に向上させる。
ビジュアル向上（Phase 3）の前にゲームロジックを完成させ、描画対象を確定させる。

## 実装順序

```
Phase 2-A: 基礎修正（未実装の穴埋め）
    ↓
Phase 2-B: コンテンツ拡充（量の拡大）
    ↓
Phase 2-C: 新メカニクス（質の深化）
    ↓
Phase 3: ビジュアル向上（レンダリング層の刷新）
```

各スコープは独立してリリース可能。A の完了が B の前提、B の完了が C の前提。

---

## Phase 2-A: 基礎修正

**ゴール:** v0.1.0 で「定義だけあり実装が欠けている」機能を完成させる。

| 項目 | 概要 |
|------|------|
| 爆発弾 (explosion_radius) | Heavy Artillery の弾が着弾時に範囲ダメージを与える |
| 貫通弾 (pierce) | High Speed の弾が敵を貫通して飛び続ける |
| ボス laser 攻撃 | Phase 2 (50-25% HP) の laser 攻撃パターン |
| EX Burst | 極太ビーム照射（2秒間、画面上部まで直線） |
| HighSpeed 解放条件修正 | Stage 7 → 実在するステージに変更 |

**詳細:** `docs/plans/2026-03-06-phase2a-foundation-fixes.md`

---

## Phase 2-B: コンテンツ拡充

**ゴール:** ゲームのボリュームを増やし、リプレイ性を高める。

| 項目 | 概要 |
|------|------|
| ステージ追加 | Stage 6〜10（第2ボスを含む） |
| 新敵タイプ | スウォーム、ファランクス、大型機 |
| ゲートバリエーション | 新ゲートペアと効果値のステージスケーリング |
| 新形態 | 1〜2種の新 MechaForm |
| スコアボーナス | ノーダメボーナス、コンボボーナス等 |

**詳細:** `docs/plans/2026-03-06-phase2b-content-expansion.md`

---

## Phase 2-C: 新メカニクス

**ゴール:** ゲーム独自のフック（差別化要素）を追加し、アクション性と戦略性を向上させる。

| 項目 | 概要 |
|------|------|
| グレイズシステム | 弾カスりでゲージ加速、リスク＆リターン |
| ジャスト TF（パリィ） | 被弾直前の形態変化で無効化＋衝撃波 |
| グロウゲート | 弾を当てるとゲート効果値が上昇 |
| ルーレットゲート | 効果が切り替わるタイミングゲート |
| デブリ（障害物） | 弾を通さない環境オブジェクト |
| 加速レーン | スコア倍率↑＋高速スクロール帯 |

**詳細:** `docs/plans/2026-03-06-phase2c-new-mechanics.md`

## Phase 3 に送る項目（ビジュアル向上）

以下はロジック実装後にレンダリング層で対応する：

- ヒットストップ（フレーム停止演出）
- 画面揺れ（Screen Shake）
- パーティクルエフェクト（撃破、着弾）
- 機体ビジュアル刷新（形態ごとのデザイン）
- 変形アニメーション
- ゲート通過演出
- スコアポップアップ演出
- 背景パララックス
