# Phase 2-B: コンテンツ拡充 — スコープ詳細

## 前提

Phase 2-A（爆発弾・貫通弾・EX Burst 等）が完了していること。
新敵タイプは爆発弾・貫通弾による形態差別化を前提としている。

---

## 1. 新敵タイプ — 3種

### 1-A. スウォーム (Swarm)

**コンセプト:** HP 1 の小型機が大群で押し寄せる。High Speed の貫通弾がないと処理しきれない。

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| enemyType | `'swarm'` | 新規追加 |
| HP | 1 | 1発で倒せる |
| ヒットボックス | 16×16px | 通常敵(28×28)より小さい |
| 接触ダメージ | 5 | 低い（1体ずつは弱い） |
| 射撃 | なし | 体当たりのみ |
| 移動 | 下方向スクロール + 横に微揺れ | sin波でゆらゆら |
| 出現パターン | 横一列 5〜8体 or V字フォーメーション | タイムラインで一括スポーン |
| スコア | 30 | 数で稼ぐ |
| クレジット | 0 | ドロップなし |

**形態との関係:**
- Standard: 1体ずつしか倒せず厳しい
- **High Speed: 貫通弾で一列まとめて貫通 → 最適**
- Heavy Artillery: 爆発で複数巻き込み可能だが発射率が低い

### 1-B. ファランクス (Phalanx)

**コンセプト:** 正面にシールドを持つ中型機。シールド部分は弾を弾くが、穴（隙間）がある。横移動しながら降りてくるため、穴の位置が変わる。

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| enemyType | `'phalanx'` | 新規追加 |
| HP | 60 | Patrol(40)より硬い |
| ヒットボックス | 36×36px | 通常敵より大きい |
| シールド構造 | 正面の上半分にシールド、下半分に隙間 | 弾がシールド部に当たると無効化 |
| 接触ダメージ | 15 | |
| 射撃 | 2.0秒間隔 | 通常の敵弾 |
| 移動 | Patrol と同様に左右移動（速度40lps） | 横移動でシールドの穴位置が相対的に変化 |
| スコア | 300 | 高い（倒すのに工夫が必要） |
| クレジット | 3〜5 | 高報酬 |

**形態との関係:**
- Standard: シールドの穴を狙って正面から撃つ（テクニカル）
- **Heavy Artillery: 爆発の範囲ダメージがシールドを迂回 → 有効**
- High Speed: 貫通弾がシールドを通過 → 有効
- シールド判定: 弾のY座標が敵の上半分に当たった場合はダメージ無効

### 1-C. 大型機 (Juggernaut)

**コンセプト:** 大きな機体に複数の砲塔を持つ重装甲機。砲塔ごとに独立して射撃する。

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| enemyType | `'juggernaut'` | 新規追加 |
| HP | 120 | ボス級の耐久力 |
| ヒットボックス | 56×48px | 通常敵の約2倍 |
| 砲塔数 | 3 | 左・中央・右 |
| 砲塔射撃 | 各砲塔が1.5秒間隔で交互に射撃 | 位相をずらして連射に見せる |
| 弾ダメージ | 12/弾 | |
| 接触ダメージ | 25 | 大型なので痛い |
| 移動 | 低速で下降（スクロールの0.3倍速） + ゆるい左右移動 | 長時間画面に滞在 |
| スコア | 500 | 中ボス級 |
| クレジット | 5〜8 | 高報酬 |

**形態との関係:**
- Standard: 地道に撃ち続ける
- Heavy Artillery: 高ATKで早く削れるが、複数弾の回避が厳しい（低速）
- **High Speed: 高発射率で削りつつ機動力で弾幕回避 → バランス良い**
- TF切替の判断: 砲塔破壊で弾数を減らすか、本体を集中攻撃するか

## 2. ステージ追加 — Stage 6〜10

### ステージ構成

| Stage | 名称（仮） | 時間 | ボス | テーマ |
|-------|-----------|------|------|--------|
| 6 | Scrap Yard | 100秒 | - | スウォーム初登場、貫通弾の有効性を学ぶ |
| 7 | Fortress Gate | 110秒 | - | ファランクス初登場、シールドの対処を学ぶ |
| 8 | War Front | 120秒 | - | 大型機初登場、全敵タイプ混合 |
| 9 | Final Approach | 130秒 | - | 高密度混合、ゲートバリエーション |
| 10 | Omega Core | 180秒 | boss_2 | 第2ボス戦 |

### 各ステージの設計方針

**Stage 6 — スウォーム導入:**
- 前半: 既存敵（Stationary, Patrol）で軽いウォームアップ
- 中盤: スウォーム初出現（横一列5体）→ 貫通弾で処理する体験
- 後半: スウォーム + Patrol の混合、ゲートで形態強化しながら対応

**Stage 7 — ファランクス導入:**
- 前半: Patrol 中心
- 中盤: ファランクス初出現 → シールドの穴を狙うか、爆発弾で回り込むか
- 後半: ファランクス + スウォーム → 形態切替が重要に

**Stage 8 — 大型機導入:**
- 前半: 各タイプの混合ウェーブ
- 中盤: 大型機が1体出現（中ボス的な扱い）
- 後半: 大型機 + 随伴のスウォーム

**Stage 9 — 総力戦:**
- 全敵タイプが登場する高難易度ステージ
- ゲートの選択がより重要に（Tradeoff が増える）
- コンボ狙いの Enhance gate 配置

**Stage 10 — 第2ボス:**
- ボス前の準備フェーズ（Enhance gate 3連でAwakened狙い可能）
- boss_2: boss_1 より攻撃パターンが増加（spread + laser + ドローン同時）
- HP: 750 (bossIndex=2 の計算結果)

## 3. ゲートバリエーション拡張

### 新ゲート定義

| ゲート名 | タイプ | 効果 | 備考 |
|---------|--------|------|------|
| GATE_ATK_UP_15 | enhance | ATK +15 | 後半ステージ用の強化版 |
| GATE_FR_UP_30 | enhance | FR +30% | 後半ステージ用 |
| GATE_HEAL_FULL | recovery | HP 100% | レアゲート |
| GATE_RAPID_GLASS | tradeoff | FR ×2.0, HP -30 | ハイリスク |
| GATE_TANK | tradeoff | HP +50, SPD ×0.7 | 防御寄り |

### 新ゲートペア

| ペア名 | 左 | 右 | レイアウト |
|--------|-----|-----|-----------|
| PAIR_ENHANCE_STRONG | ATK+15 | FR+30% | forced |
| PAIR_RECOVERY_FULL | HP 100% | ATK+10 | forced |
| PAIR_TRADEOFF_EXTREME | Rapid Glass | Tank | optional |
| PAIR_ATK_FR | ATK+5 | FR+20% | forced |

### ゲート効果のステージスケーリング

Phase 2-B ではゲート定義を静的に増やす方針で進める。
動的スケーリング（stageId に応じて効果値が変わる）は Phase 2-C 以降で検討。
